CSS的选择器

1.选择器定义

选择器是CSS规则的第一部分。它是元素和其他部分组合起来告诉浏览器哪个HTML元素被选择, 并且该元素被应用选择器后大括号中的CSS规则。选择器所选择的元素，叫做选择器的对象

2.选择器列表

对于样式相同的元素，可以将它们的选择器合并，即用逗号隔开多个选择器，这就 形成了一个选择器列表，这个列表中的所有选择器对应的元素使用列表括号后的相同样式。例子如下：

我是lizibox（class），我需要使用蓝色

我是lizih3（id），我也需要使用蓝色

上述例子使用.lizibox, #lizih3{}列表来进行相同样式处理,实际使用时可以在逗号后换行，这样更能分辨列表中选择器数量

注意:如果列表中有一个选择器出现错误，整个列表都会失效，其他选择器也会被忽略

3.选择器种类

类型、类和ID选择器

即常见的三种选择器：

  • 元素选择器
  • 类选择器(class)
  • id选择器(id)

标签属性选择器

  • 根据一个元素上的标签属性是否存在以选择：
    a[title] { }
    img[alt] { }
  • 根据一个元素上的标签属性是否有特定值来选择：
    a[href="https://www.example.com"] { }

伪类与伪元素选择器

  • 伪类：元素的一个特定状态，选择元素后使用:status来表示元素在此状态下的样式，如：

    a:hover { } /* :hover伪类表示鼠标悬浮在选择的类上时的样式*/

    具体例子如下：

  • 伪元素：选择元素的一部分而不是整个元素，选择元素后使用::part来表示元素的一部分，如：

    p::first-line { } /* ::first-line伪元素表示p元素的第一行 */

关系选择器

通过运算符将各种选择器组合起来，形成关系选择器

  • 后代选择器：使用空格即可
    article p { }  /* 选择了article元素内的所有p元素 */
  • 子代选择器：使用>
    article > p { }  /* 选择article元素内的初代子元素
  • 相邻兄弟选择器：使用+
    #idname + p { }   /* 选择idname元素紧跟的p元素
  • 通用兄弟选择器：使用~
    h1 ~ p {}   /* 选择h1元素之后的所有p元素，只需同级不需紧跟
  • 通配选择器：*，可以匹配任意类型HTML元素
    *[title^=CSSSelector] {}  /* 选择title不等于CSSSelector的元素
注:不建议使用通配选择器
