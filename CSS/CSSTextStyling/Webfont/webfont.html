web字体

1.web字体解决的问题

对于传统字体，在字体堆栈中选择系统中可用的字体是一般方式。但是，只有少数几种字体在所有系统中都兼容。 这就导致了开发者只能使用web-safe字体以及系统默认字体，字体使用不方便。

2.web字体

web字体是一种CSS特性，允许访问网站时一起下载字体文件；在CSS开头使用@font-face指明字体名称和url即可： 

              @font-face {
                font-famliy: "my-font";
                src: url(fonts/my-font.tff);
              }

使用时，直接使用自定义名称"my-font"即可

3.在线字体服务

使用在线字体服务，如谷歌字体，只需要将谷歌提供的字体资源代码插入html和css即可
