链接文本样式

1.链接的基本样式

链接有很多默认的样式，如链接颜色，链接状态等

链接状态

链接的每个状态可以是用选择器的伪类选择器进行选择

  • link:未访问过的，链接默认状态，用伪类:link来应用样式
  • visited:链接已被访问的状态，存在与浏览器历史纪录中，使用:visited伪类来应用样式
  • hover:鼠标停留在链接上，使用:hover伪类来应用样式
  • focus:链接被选中（如用tab键移动到链接）或者使用js的HTMLElement.focus()选中时， 可以使用:focus伪类来应用样式。（不是指鼠标点击时的状态）
  • active:链接被激活时（如被点击时候），使用:active伪类来应用样式

链接的基本样式

  • 具有下划线
  • 未访问过的:link是蓝色的

  • 选择链接时:focus链接周围有一个轮廓
  • 激活链接时（点击时）:active，链接会变成红色

可以使用诸如color,text-decoration,outline,cursor 的属性样式化链接。其中:

  • outline,表示文字的轮廓，与border不同的是它不占用盒模型空间，它只是设置在背景图片顶部
  • cursor,表示鼠标光标的样式，可以使用关键字，也可以使用url指定

链接伪类的写法顺序

编写链接样式需要对链接元素进行选择，应该依次如下选择:

  1. a { }
  2. a:link { }
  3. a:visited { }
  4. a:focus { }
  5. a:hover { }
  6. a:active { }

这样的顺序是有意义的，第一位是a{}显然是对的； 然后的样式化状态应该是未访问前后访问后的静态状态。 至于为什么active状态在最后，因为一个链接被激活时，它已经被鼠标覆盖和选中，所以需要提前编写hover状态
