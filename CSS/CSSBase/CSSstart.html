CSS

1.CSS定义

CSS:层叠样式表(Cascading Style Sheets)，用于设置和布局网页，例如更改内容的字体，颜色，大小，间距等.
将内容拆分为多个列，或添加动画和其他装饰功能。

从实用角度来说：

  1. 浏览器会解析HTML标记语言
  2. 浏览器对不同的语义标签有一套默认的显示样式（字体大小，间距等）
  3. 网页需要更多样性的风格，不然会很枯燥
  4. CSS就是用来控制浏览器如何显示HTML元素的规则集，用于创造自己喜欢的样式

4.CSS规范

一些组织和公司对CSS如何书写提出的标准规范。CSS标准会不断更新，但肯定会一直兼容旧内容。

不同浏览器对CSS有不同兼容性。当一个浏览器支持 CSS 后，我们就可以用它来进行Web开发了。 这意味着我们写在 CSS 文件中的代码可以被指令执行，展示到荧幕中。 但是让所有的浏览器都同时支持一个 CSS 新特性是不现实的，通常都会一个空档期 — 一些浏览器已经支持而另一些仍未支持。
