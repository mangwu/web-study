3.CSS代码结构

  • CSS规则集：n个CSS规则
  • CSS规则：选择器 + { + CSS声明块 + }
  • 声明块：n个CSS声明;
  • CSS声明：属性: 属性值。（核心）

4.CSS的函数

CSS中大部分属性值都是关键字或数值，但是有些属性值可以是函数

  • calc()函数，用于简单计算，得到数值结果作为属性值
    例如，在width属性值中使用这个函数，根据父元素width计算需要的属性值
    width="calc(90%-30px)"
  • transform属性：属性值用函数(transform-function)表示，属性用于对元素进行旋转，缩放，倾斜，平移
    使用rotate()函数，对元素进行顺时针旋转，参数为0-1.0turn
    0.6turn


      transform属性值的表示方法（transform-function）：
      对原图进行一些列转换：
    1. matrix()函数：

      格式：matrix(a,b,c,d,tx,ty)，矩阵表示为：

      a c tx
      b d ty
      0 0 1
      参考 matrix()
      x' = ax + cy +tx
      y' = bx + dy +ty

      1. 平移只与tx，ty有关系。
        HTML默认左上角为坐标原点,x正轴水平向右，y正轴水平向下
        沿着x轴正方向平移30个单位(px)沿着y轴正方向平移40个单位(px)
        transform:matrix(1, 0, 0, 1, 30, 40)
      2. 缩放：根据式子，x'在x轴方向大小与a有关系
        而y'在y轴方向大小与d有关系，a,d代表在x或y轴的缩放倍数
        x轴方向放大1.3倍,y方向变为0.8倍
        transform:matrix(1.3, 0, 0, 0.8, 0, 0)
      3. 旋转：按照旋转角度对a,b,c,d四个进行赋值:
        a=cosθ，b=sinθ,c=-sinθ,d=cosθ,默认旋转方向为顺时针
        cosθ -sinθ 0
        sinθ cosθ 0
        0 0 1
        x'=xcosθ - ysinθ, y'=xsinθ + ycosθ
        θ=30°,顺时针旋转30°
        tx=223,左移223单位
        transform:matrix(cos30|0.866,sin30|0.5,-sin30|-0.5,cos30|0.866)
    2. rotate()函数：旋转函数，指定旋转角度，正数为顺时针旋转，负数为逆时针旋转
      1. deg单位，使用角度制：60deg就是60度
        θ=30deg,顺时针旋转30deg
        tx=223,左移223单位
        transform: matrix(1, 0, 0, 1, 223, 0) rotate(30deg)
      2. turn单位,turn表示圈，1turn等于360deg，60deg=60/360deg=1/6turn
        θ=30deg,顺时针旋转30deg,1/12turn
        tx=223,左移223单位
        transform: matrix(1, 0, 0, 1, 223, 0) rotate(0.083)
    3. scale()函数：缩放函数，缩放由矢量定义，scale(sx,sy=sx)
      sx=1.5 sy=0.9
      x轴方向放大1.5倍，y轴方向缩小置0.9倍
      只设置sx，sy默认等于sx
      参数为负数会进行沿轴对称处理
      transform: scale(1.5,0.9);
    4. skew()函数：拉伸函数，skew(ax,ay),表示沿着坐标扭曲元素的角度
      ax=10deg ay=10deg
      x轴正方向拉伸10°，y轴正方向拉伸置10°倍
      参数为负数会进行沿负方向轴拉伸处理，若ax=-ay，相当于旋转
      transform: skew(10deg,10deg)
    5. translate()函数:平移函数，tarnslate(tx,ty),tx,ty表示平移单位数
      tx=80px ay=20%
      x轴正方向平移80px，y轴正方向平移自身宽度的20%
      参数为负数会进行沿负方向轴平移处理
      注意matrix中tx,ty平移没有单位，这个可以是多种不可省略单位(em,px,ex,%)
      transform: translate(80px,20%);
