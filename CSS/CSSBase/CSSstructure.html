3.CSS代码结构

  • CSS规则集：n个CSS规则
  • CSS规则：选择器 + { + CSS声明块 + }
  • 声明块：n个CSS声明;
  • CSS声明：属性: 属性值。（核心）

CSS属性设置为特定属性值是CSS功能核心。属性和属性值区分大小写

      1. larger:我是例子(这种大小应该是large大小）
      2. smaller:我是例子(这种大小应该是small大小）
    3. 长度值大小：单位px和em或ex
      1. em：相对于父元素文字的大小。 0.5em就是当前元素的父元素的字体大小的一半
      2. ex：ex 是指相对于父元素所用字体中小写 x 的高度，通常为字体高度的一半。 1.0ex就是当前元素的父元素的字体大小的一半
      3. px:不会随屏幕分辨率改变的固定大小值。 我是字体大小为26px的例子
    4. 父元素大小的百分比值（%） 50%父元素大小的例子
    5. 推荐使用方式：使用相对值大小和长度值大小；理由如下:
      1.由于设备的不同，绝对值大小（large等）会随着操作系统所认为的屏幕分辨率的大小（通常是不对的）改变而改变
      2.最好使用用户默认字体大小的相对大小
      3.em和ex是相对于默认字体大小的单位，推荐使用
      4.px虽然是绝对值大小，但它的大小不会随屏幕分辨率大小而改变
    6. 技巧：
      一个流行的技巧是设置body元素的字体大小为62.5% (即默认大小16px的62.5%)，等于10px。
      现在你可以通过计算基准大小10px的倍数，在任何元素上方便的使用em单位。
      这样有6px = 0.6em, 8px = 0.8em, 12px = 1.2em, 14px = 1.4em, 16px = 1.6em。

  • background-color:设置元素背景色，默认值为transparent，不是继承元素。属性值如下
    1. 关键字：如red，green等
      我的背景颜色是红色
    2. 16进制颜色，格式：#000000 - #ffffff
      我的背景颜色是 #789abc
    3. RGB颜色，格式：rgb(0-255, 0-255, 0-255,opacity) （最后一项为透明度）
      我的背景颜色是(123, 231, 213,0.5)
    4. HSLA颜色，
      HSL色彩空间中，H（Hue）：代表色调，S（Saturation）：代表饱和度，L（Lightness）：代表亮度，（A（Alpha）：代表不透明度）。
      H是色度，取值在0度~360度之间，0度是红色，120度是绿色，240度是蓝色。360度也是红色。
      S是饱和度，是色彩的纯度，是一个百分比的值，取值在0%~100%，0%饱和度最低，100%饱和度最高
      L是亮度，也是一个百分比值，取值在0%~100%，0%最暗，100%最亮。
      A是不透明度，取值在0.0~1.0，0.0完全透明，1.0完全不透明。

      亮度下降:
      hsla(120, 100%, 80%, 0.5)
      hsla(120, 100%, 70%, 0.5)
      hsla(120, 100%, 60%, 0.5)
      hsla(120, 100%, 50%, 0.5)
      hsla(120, 100%, 40%, 0.5)

      透密度下降：
      hsla(120, 100%, 60%, 1.0)
      hsla(120, 100%, 60%, 0.8)
      hsla(120, 100%, 60%, 0.6)
      hsla(120, 100%, 60%, 0.4)
      hsla(120, 100%, 60%, 0.2)
    5. 特色关键字：
