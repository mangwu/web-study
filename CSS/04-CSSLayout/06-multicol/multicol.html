column-width

column-width设置列宽度，按照设置的列宽度尽可能多的创建列，然后剩余空间会被列平方： 无法期望宽度正好是设置的列宽度，除非容器宽度正好整除指定宽度

.column-width { column-width: 200px; }

2.列样式

列样式在容器中统一改变,无法给单独的列创造样式

  • column-gap: 改变列间间距
  • column-rule: 加入分割线，输入值和border类似，分割线不占用宽度
.column-style {
  column-gap: 20px;
  column-rule: 5px dotted dodgerblue;
}
