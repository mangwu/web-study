# 组件化开发

## 类比

+ 处理大问题
  + 逻辑能力有限
  + 不太可能一次解决一个大问题
  + 将问题进行拆解
  + 解决很多小问题
  + 大问题也会迎刃而解

## 组件化

+ 开发一个大网页
  + 将以一个页面所有处理逻辑放在一起会很复杂
  + 不利于管理与维护
  + 将页面拆分成一个个小功能块
    + 每个组件可以再次划分
  + 每个独立功能块结合在一起
  + 便于管理和维护

### Vue组件化思想

+ 提供一种抽象，让我们开发独立可复用的小组件构造应用
+ 任何的应用都会被抽象成一颗组件树
+ 尽可能将页面拆分成一个个小的、可复用的组件
