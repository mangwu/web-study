1.条件语句

条件语句结构允许JS在代码执行时做出执行选择，从而获得不同结果

if……else语句

最常见的条件语句，其代码块结构如下：

if (condition) {
 code to run if condition is true
} else {
 run some other code instead
}

else if语句

如果不止两个选择可以使用else if连接更多的选择

if if else else

嵌套，在if语句块中嵌套另一个if……else语句是完全可行的

switch语句

当选择过多时，可以使用switch语句替代if……else条件语句，注意不要忘记写break!,因为switch会依次判断执行 其代码结构如下：

switch (expression) {
 case choice1:
  run this code;
  break

 case choice2:
  run this code;
  break
// 许多cases
default:
 没有一个case符合情况时，运行
}

使用switch语句完成

三元运算符

使用三元运算符可以实现简单的两个选择项，例如黑白主题选择；三元运算符格式：

(expression) ? true code : fasle code;

循环

一个循环需要的条件：

  • 一个开始条件：开始条件被初始化为一个特定值，值满足条件就开始循环，循环的起点
  • 一个结束条件：循环停止的标准-通常为计数器达到一定值
  • 一个迭代器：通常在每个连续循环上递增少量的计数器，直到达到结束条件

for循环语法

for (初始化器; 退出条件; 迭代器) {
 code to run

break语句

break语句用于跳出循环，在switch语句中每个case都会使用；在for循环中在配合if语句使用，如果满足条件就调查循环； 常用于查找，例如在用for循环遍历一个数组时，如果找到了就可以直接跳出，避免资源浪费。 下面是一个在固定数组中查找联系人电话的搜索例子,找到就直接使用break退出
