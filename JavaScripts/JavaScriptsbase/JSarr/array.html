介绍：数组的基本理解

1.数组

数组：像列表一样的对象。数组是包含多个值的对象，数组对象存储在变量中， 与普通变量的区别是可以通过方括号访问列表中的每个值。

数组声明

数组由方括号构成，用逗号分隔元素列表;每个项可以是不同数据类型。

let arr = [1, 2, 'acc', [2, 'asc', 5]];

修改数组元素

使用方括号和索引访问修改数组项内容

获取数组长度

通过length属性获取数组长度,通过数组长度可以遍历数组

arr.length;

数组方法

数组方法一般使用句点调用法，可以对数组进行添加删除等，也有可以使字符串拆分为数组的方法

字符串和数组之间的转换

split()方法： 字符串句点调用，参数为将字符串分隔的字符，返回子字符串，作为数组中的项

let mydata = '1 kk 254 asc as';
let myarr = mydata.split(' ');// 以空格分隔，结果为['1','kk','254','asc','as']

join()方法： 数组句点调用，参数为要插入每个数组项之间的字符，合成返回一个字符串； 嵌套数组中的间隔仍然是逗号，不受join中参数的影响

let mystring = myarr.join(','); //以逗号合并，结果为'1,kk,254,asc,aas'

toString()方法： 数组句点调用，无参数，自动使用逗号作为合并的分隔字符，返回合并字符串。 注意其对嵌套数组也有效，因为嵌套数组转换成字符串时，数组项之间间隔默认用逗号隔开

let mystring2 = myarr.toString(); //逗号合并，结果为'1,kk,254,asc,as'

添加或删除数组项

数组是可变数据对象，可以用方法当场修改（字符串不行，使用方法只能返回修改后的字符串)

使用push()和pop()方法添加和删除数组项; push()可以使用多个参数，添加多个数组项，并且返回修改后的数组长度； pop()不需要参数，直接删除数组最后一个数组项，并返回被删除项

let newlength = myarr.push('123', 4); // myarr : ['1', 'kk','254','asc', 'as','123',4]
newlength; // 7
let rmove = myarr.pop(); // rmove = 4; myarr.length == 6

使用unshift()和shift()添加和删除数组项，作用于数组开始而不是结尾

newlength = myarr.unsift('4'); //myarr: ['4', '1', 'kk','254','asc','as','123']
newlength; // 7 rmov = myarr.shift(); //rmove = '4'; myarr.length == 6
