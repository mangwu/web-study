1.数字

数字类型

  • 十进制：
    • 整数，如10， -5
    • 浮点数，有小数点或小数位，如1.25，-1.0
    • 双精度，特定类型的浮点数，有更高的精度
  • 二级制
  • 八进制
  • 十六进制：在CSS中可以在颜色值时使用

JS只用Number类型表示数字类型（interger和decimals）。

JS中的全部数字

定义变量存储数字，用typeof查看数字类型，验证JS数字都用一Number类型

let myInt = 5;
let myFloat = 6.61;
typeof myInt;
typeod myFloat;

2.运算符

运算符包括算式运算符、自增减运算符、赋值运算符、比较运算符

算式运算符

  • + - * / 加减乘除
  • % 取模，8 % 3 = 2
  • ** 幂, 2**3 = 8
    Math.pow()方法也可以完成指数运算，如Math.pow(2,3) == 2**3

优先级
** > % == * == / > + == -

自增自减运算符

  • ++ 变量值自增1；放在变量后，返回变量值再自增；放在变量前，自增再返回变量值； 不能运用于数值
  • -- 变量值自减1；放在变量后，返回变量值再自减；放在变量前，自增再返回变量值； 不能运用于数值
let myInt = 5;
console.log(myInt ++) // 5
console.log(++ myInt) // 7

赋值运算符

  • = :将右边的值赋给左边的变量
  • +=: 加法赋值
  • -=: 减法赋值
  • *=: 乘法赋值
  • /=: 除法赋值
  • <<=: 左移位运算符
  • >>=: 右移位运算符
  • &=： 按位与赋值
  • ^=: 按位异或赋值
  • |=： 按位或赋值

比较运算符

  • === 严格等于，比较值和数据类型
  • == 等于，只比较值
  • !== 严格不等于，比较值和数据类型
  • != 不等于，只比较值
  • > < >= <= 大于 小于 大于等于 小于等于

3.逻辑运算符

包括位逻辑运算符、移位运算符（这二者又称为位运算符）、逻辑运算符

  • 位逻辑运算符：
    • &: 位与 4 & 5 == 4 ; 100 & 101 = 100
    • |： 位或 13 | 7 == 15 ; 1101 | 0111 = 1111
    • ^: 异或 13 ^ 7 == 10 ; 1101 ^ 0111 = 1010
    • ~: 位非 ~13 == -14 ; ~0000……1101 = ~1111……0010
  • 移位运算符:
    • << 左移；左移指定位数，右边补0； 9<<2 == 36
    • >> 右移；带符号右移指定位数，左边补位为符号位； -9 >> 2 == -3 ； 1111……0111 >> 2 == 1111……1101
    • >>> 右移:不带符号右移指定位数，左边补位0； -1 >>> 1 == 2147483647; 1111……1111 >>> 1 == 0111……1111
  • 逻辑运算符：
    • &&： 逻辑与；表达式两端都为true才是true
    • ||:逻辑或；一个为true就是true
    • !: 逻辑非；将true和false反转
     关于flase值，可以被转为flase的值有null、0、NaN、 ''和undefined

4.其他运算符

  • （条件）三元运算符
    条件?:值1:值2
    var status = (age >= 18)? 'adult':'minor';
  • 逗号运算符 利用逗号同时对表达式求值返回。如用逗号操作符同时改变两个变量的值：
    var x = [0,1,2,3,4,5];
    var a = [x, x, x, x ,x];
    for (var i = 0, j = 5; i <= j; i++, j--) console.log(a[i][j])
  • 一元操作符 操作一个操作数：
    • delete: 删除一个对象属性或者一个数值中的某个键值,注意删除数组某一个值时，值变为undefined，但是数组元素仍然存在
      delete objectName.property;
      delete objectName[index];
      deletex = 42;
      let deletey = 43;
      var deletez = 43;
      var deletearr = [2, 5, 6, 7];
      console.log(delete deletex); // true
      console.log(delete deletey); // false
      console.log(delete deletez); // false
      delete deletearr[2];
      console.log(deletearr[2]); //undefined
    • typeof :使用方式 typeof operand; 或 typeof (operand);返回关键字数据类型
    • void : 使用方式 void expression; 或 void(expression);表明一个表达式没有返回值。 expression是javaSciript表达式
      • 创建一个超链接，使用void指明没有返回链接，点击时不会有任何效果：
        <a href="javascript:void(0)">点击这里什么都不发生</a>
        点击这里什么都不会发生
      • 创建一个超链接，使void中的表达式执行具体操作，但是不会返回链接，如执行提交表单的操作：
        <a href="javascript:void(document.form.submit())">点击提交表单</a>
        点击提交表单
  • 关系操作符
    • in操作符：指定属性确实存在所指定对象就会返回true
      • 对于数组，可以使用索引 in arr;表明数组中是否有该索引值，不能使用值进行;
        var trees = new Array('redwood', 'bay','cedar');
        0 in trees; // true
        'bay' in trees; // false，无法使用值进行判断
        'length' in trees; // true length是数组的一个属性
      • 对于预定义的对象，可以对对象中存在的属性进行判断
        'PI' in Math; // true
      • 对于自定义对象，可以对自己定义的属性进行判断
        var myBike = {make:'lucs', model:'a5', year:2015};
        'model' in myBike; // true
    • instanceof 判断对象是所指的类型，返回true;形式如下：
      objectNmae instanceof objectType; // 返回布尔值
