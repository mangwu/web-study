1.字符串

创建一个字符串如下：

let string = '创建的字符串内容';

单引号和双引号

根据个人喜好使用单或双引号。可以在单引号中使用双引号，也可以在双引号中使用单引号，但不能使用同一种(除非使用转义字符)

let string1 = "I'm Alice";
let string2 = 'He said:"howdy"'

转义字符

使用\来转义特殊符号为文本，如在带引号字符串中使用\'来使这个单引号没有结束意义， 变成纯文本。下面是转义字符表：

转义字符表
Code Output
\0 空字符
\' 单引号
\" 双引号
\\ 反斜杠
\n 换行
\r 回车
\v 垂直制表符
\t 水平制表符
\b 退格
\f 换页
\uXXX unicode码

2.连接字符串

使用+符号连接字符串

  • 纯字符串连接：
    let string = 'a' + 'b'; //'ab'
  • 变量串联：
    let name = prompt('input name');
    alert('hello' + name + 'nice to meet you')
  • 字符串和数字连接:可以直接使用+号连接，返回字符串：
    let name = 'mangwu' + 2000; // output 'mangwu2000'
  • 字符串与数字转换: Number()和toString()
    let string = '123';
    string = Number(string); // 123
    let mynum = 123;
    mynum = mynum.toString(); // '123'

3.字符串方法

把字符串当作对象；javascript中的一切都是对象；对变量赋值字符串相当于变量成为了字符串的一个实例

字符串长度

length属性，使用句点法查看;如string.length;

检索特定字符串字符

使用方括号返回字符串中的任意字符,索引从0开始

let string = 'abcde';

console.log(string[2]) // 'c'

查找子字符串

indexof()方法，使用字符串句点调用方法，需要一个字符串作为参数， 返回值为子字符串在父字符串的开始索引,没有就返回-1.

let string = 'abcdefg';
string.indexOf('def'); // 返回值3

切片

slice() 方法，第一个参数为字符串切割开始索引，第二个参数为字符串切割结束索引，可以省略，默认为父字符串末尾。 可以通过查找字符串和切割方法切割出想要的字符串。

let string = 'abcdefg'; let stringcut = string.slice(2, 5);

切换大小写

toLowerCase()和toUpperCase()将所有字符串转换成小写或大写。

let Data = 'I am Alice';
Data.toLowerCase(); // i am alice
Data.toUpperCase(); // I AM ALICE

替换字符某一部分

repalce()方法，将字符串中的一个子字符串替换成另一个字符串。 第一个参数为要被替换的子字符串，第二个为换上的字符串。

let string = 'abcdefg';
string.replace('cd', 'CD');
