1.变量

变量就是一个存储数值的容器。 这个容器可能是一个用于累加的计算的数字，或者一个句子中的字符串。 变量最重要的特性就是：

存放的数值是可以改变的

变量另外一个特性就是：

可以存储任何东西，甚至是函数

除此之外，需要理解的是变量不是数值本身，而是一个存储数值的容器。相当于一个盒子。

2.变量声明

声明变量即创建变量，声明是更准确的说法。使用var和let关键字后加上变量名和分号:

let myVariable1;
var myVariable2;

在浏览器console窗口输入变量名，会返回undefined的返回值。 因为变量还没有被赋值。

初始化变量

初始化变量语句结构如下:
  • （如果变量未声明)var or let
  • 空格
  • 变量名
  • 空格
  • =
  • 空格
  • 值
  • ;
var myVariable = 'value';

var和let的区别

最初的JS只有var，因为var有一些局限性， 就创建了一个新关键字let，用来创建一些不适合作为var变量的 或者说一个工作方式不同的变量。

  • var有变量提升特性而let没有; 可以先使用变量赋值，后对变量进行声明
    myVariable = 'value';
    var myVariable
  • var可以对相同名称的变量进行多次声明，但是let不行，以下结果会报错
    let myVariable = 'value';
    let myVariable = 'value'; /* 报错行，使用var不会报错 */
  • var声明的变量的作用域是全局的，而let声明的变量只在声明的块或子块中可以使用
    {
     var x = 1;
     {
      var x = 2; //同样的变量
      console.log(x); //output 2
     }
     console.log(x); //output 2
     let y = 1;
     {
      let y = 2; //不同的变量
      console.log(y); //output 2
     }
     console.log(y); //output 1
    }
  • 使用let而非var

变量命名规则

  • 使用0-9，a-Z和下划线
  • 变量名不要以下划线和数字开头，最好以小写字母开头(数字开头是错误的，下划线开头的含有特殊意义)
  • 小写驼峰命名法： 如lookLikeThis
  • 大小写敏感
  • 避免使用保留字

3.变量类型

  • Number:
    包含整数，浮点数。不用声明类型，直接赋值即可:
    let myAge = 21;
  • String
    字符串，用单引号或双引号包起来
    let myAge = '21';
  • Boolean
    布尔值，true 或者 false
    let isAlive = true;
    let test = 6 < 3;
  • Array
    数组，包含很多值，用方括号包围，用逗号隔开
    let arr = ['alice', 'async', 'defer']; arr[1]; // 访问数组，返回值'async'
  • Object
    对象，和python中的字典很象,但是访问方法不同，使用句点访问法，且键就是实体
    let dog = [name:'xiaohu', weight:'23', action:'sitting'];
    dog.name; //返回'xiaohu'

4.动态类型语言

JavaScript是动态类型语言，不需要指定数据类型（python也是这样）

使用typeof查看变量存储的数据的类型。

let myNumber = 5.166;
typeof myNumber;
